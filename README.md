# norminette.cr

Crystal implementation of the official 42 Norminette.

## Installation

```
git clone https://github.com/aabajyan/norminette.cr
cd norminette.cr
crystal build src/cli.cr -o norminette
alias norminette='/path/to/norminette'
```

## Usage

```
norminette [/path/to/project]
```
